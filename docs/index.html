http

A basic HTTP web-server written in C++ using the Linux sockets API. The source code can be found on the GitHub repository.

To get started, clone the repository in your local machine, and import the http/include/server.hpp header in your C++ file. Following is the documentation for the classes that can be used for creating and running the server. All classes and functions are declared under the http namespace.
